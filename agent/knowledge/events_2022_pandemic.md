# 2022年疫情封控对各省经济的冲击

## 上海封城：3-6月全面封控

2022年3月底至6月初，上海实施全面封控管理，影响深远：

- 上海二季度GDP同比下降13.7%，为历史最大单季跌幅
- 全年GDP增速仅-0.2%，是改革开放以来罕见负增长
- 上海港运转受阻，全球供应链受影响（上海港集装箱吞吐量全球第一）
- 半导体产业冲击严重：全国约25%的半导体产业集中在上海及周边
- 4月全国整车货运流量指数同比下降超25%

## 全国封控影响范围

据研究机构统计，2022年全国共有152个城市在不同时期实施部分或全面封控措施，影响人口超过2.8亿。

- 二季度全国GDP仅增长0.4%，几乎停滞
- 11月疫情防控政策突然转向（"二十条"和"新十条"），社会预期剧烈波动
- 社会消费品零售总额全年下降0.2%，消费遭受重创
- 服务业生产指数多个月份同比负增长

## 对本实验结果的影响

2022年是本实验6年中聚类结构发生根本变化的一年：

1. **Silhouette系数降至0.41**（6年最低），说明省份间边界模糊化
2. **京沪从"第一梯队"降入"第二梯队"**：封控导致上海经济严重受损，北京也受严格防控影响
3. **广东升入"第一梯队"**：虽然广东也受外贸下降影响（增速仅1.9%），但总量优势使其相对排名提升
4. **TOP5分数异常接近**：广东100、上海90.99、江苏90.51、浙江89.56、北京89.30，差距被压缩到10分以内
5. **收入权重升至0.1695**（6年最高）：经济下行期，居民收入水平的省际差异区分度更大

## 吉林省的特殊影响

吉林省2022年疫情严重，GDP增速下降1.9%（全国最低）。长春市实施长时间封控，汽车产业链中断。但这也为2023-2024年吉林的强劲反弹奠定了低基数效应。
