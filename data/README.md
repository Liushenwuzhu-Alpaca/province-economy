# 数据目录说明

本目录存放项目的原始数据文件。代码接口文档见 `src/data/README.md`。

---

## 数据覆盖

2019-2024 共 6 年，31 个省/自治区/直辖市，15 个经济指标。

## 目录结构

```
data/
├── raw/
│   ├── nbs_exports/      # 国家统计局导出的官方 CSV（主数据源）
│   ├── ocr_outputs/      # 整理后的各省完整指标表（供程序读取）
│   ├── yearbook_2023/    # 2023 年统计年鉴 JPG（已 gitignore）
│   ├── yearbook_2024/    # 2024 年统计年鉴 JPG（已 gitignore）
│   └── archive/          # 早期测试文件
├── results/              # 分析结果缓存（已 gitignore，可重算）
```

> `data_cache/` 和 `data/results/` 已加入 `.gitignore`，可从原始数据重新生成。
