# 指标体系与分析方法

## 指标体系

本项目采用10个核心指标，分为3个维度：

### 维度1：经济实力（4个指标）
- GDP（地区生产总值，亿元）— 正向，越大越好
- GDP增速（%）— 正向，由GDP指数(上年=100)换算得到
- 社会消费品零售总额（亿元）— 正向
- 地方一般公共预算收入（亿元）— 正向

### 维度2：居民生活（3个指标）
- 居民人均可支配收入（元）— 正向
- 居民人均消费支出（元）— 正向
- 居民消费价格指数CPI（上年=100）— 负向，转换为|CPI-100|后处理

### 维度3：产业结构（3个指标）
- 第三产业占GDP比重（%）— 正向
- 固定资产投资增速（%）— 正向
- 失业率代理指标（%）— 负向，由城镇登记失业人数/(城镇就业人员+失业人数)*100计算

## 熵权法（方法A）

回答问题："哪个省排名更高？"

计算步骤：
1. Min-Max归一化到[0,1]，负向指标做反向处理
2. 计算比重矩阵 p_ij = x_ij / sum(x_ij)
3. 计算信息熵 e_j = -k * sum(p_ij * ln(p_ij))
4. 计算差异系数 d_j = 1 - e_j（差异越大权重越高）
5. 归一化为权重 w_j = d_j / sum(d_j)
6. 加权综合得分 S_i = sum(w_j * x_ij) * 100

特点：完全客观，根据数据自身的离散程度自动分配权重，无需主观判断。

## PCA + K-Means聚类（方法B）

回答问题："哪些省份的发展模式相近？"

计算步骤：
1. Z-Score标准化
2. PCA降维到2个主成分（去除指标间共线性）
3. K-Means聚类（k=4）
4. 按各簇综合水平降序重排标签（0=最高）
5. Silhouette系数评估聚类质量

四个梯队含义：
- 第一梯队（发达型）：北京、上海等
- 第二梯队（领先型）：江苏、浙江、广东等
- 第三梯队（中坚型）：各省中等发展水平
- 第四梯队（追赶型）：发展中地区

## 数据来源

所有数据来自国家统计局（NBS）公开数据：
- GDP、三次产业、GDP指数：分省年度数据
- 收入、消费支出：2016-2024
- CPI分省：2016-2024
- 零售总额、财政收入、投资、就业：2016-2024

覆盖年份：2019-2024共6年，31个省/自治区/直辖市。
